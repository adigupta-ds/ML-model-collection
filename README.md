# Machine Learning Algorithms Showcase

A collection of machine learning projects and models built using various algorithms and techniques. This repository serves as a learning resource and portfolio demonstrating practical applications of machine learning across different problem domains.

## Objectives

- Explore and implement various machine learning algorithms.
- Compare model performance on different datasets.
- Practice feature engineering and data preprocessing.
- Understand model evaluation and optimization techniques.
- Build a structured portfolio of machine learning projects.

## Algorithms Covered

### Supervised Learning
- Linear Regression
- Logistic Regression
- Decision Tree
- Random Forest



### Unsupervised Learning
- K-Means Clustering
- Principal Component Analysis (PCA)

### Time Series Forecasting
- ARIMA
- SARIMA
- Prophet
- LSTM
